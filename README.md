# PubmedCrawler
PubMed Crawler is designed to streamline literature searches for researchers and clinicians.

# What it does

Search PubMed using custom terms (genes, diseases, methods, and more)

Retrieve up to the 300 most recent relevant publications

# For each paper, the app returns:

- PubMed ID

- Title

- Authors

- Abstract text

- Journal

- Year of publication

- DOI

# Export options

Download results as XLSX or XML for easy integration into your workflows, reviews, or reports.

This tool aims to make literature exploration faster, more reproducible, and easier to integrate with downstream analyses.
